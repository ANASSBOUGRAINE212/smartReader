## How It Works Now

```
┌─────────────┐
│   Pi Server │
│             │
│ 1. Capture  │
│ 2. OCR      │
│ 3. Generate │──┐
│    Audio    │  │
└─────────────┘  │
                 │ Audio file saved
                 │ to audio/ directory
                 ↓
┌─────────────────────────────┐
│ Response to Mobile App      │
│                             │
│ {                           │
│   "text": "Bonjour...",     │
│   "audioUrl": "/api/audio/  │
│               abc123.mp3"   │
│ }                           │
└─────────────────────────────┘
                 │
                 ↓
┌─────────────────────────────┐
│ Mobile App                  │
│                             │
│ 1. Receives response        │
│ 2. Constructs full URL:     │
│    http://192.168.X.X:5000/ │
│    api/audio/abc123.mp3     │
│ 3. Downloads audio          │
│ 4. Plays on phone 🔊        │
└─────────────────────────────┘
```

## Troubleshooting

### Audio still not playing?

**Check 1: Server restarted?**
```bash
# Stop old server (Ctrl+C)
# Start new server
python -m src.app
```

**Check 2: Audio files exist?**
```bash
ls -lh audio/
# Should show .mp3 files
```

**Check 3: Endpoint working?**
```bash
# Test audio endpoint (replace with actual filename)
curl -I http://localhost:5000/api/audio/test.mp3
# Should return: HTTP/1.1 200 OK or 404 if file doesn't exist
```

**Check 4: Network connection?**
- Verify phone and Pi on same network
- Check connection status: 🟢 Connected

### New scan but no audio?

**Possible causes:**
1. gTTS failed to generate audio
2. No internet connection (gTTS needs internet)
3. Text is empty

**Check server logs:**
```bash
# Look for TTS errors
# Should see: "TTS synthesis complete: audio/abc123.mp3"
```

### Audio plays but sounds wrong?

**Check language setting:**
1. Go to Settings tab
2. Verify language is correct (French/English)
3. Adjust speech rate if too fast/slow

## Benefits of Phone Audio Playback

✅ **Better speakers** - Phone has better audio quality than Pi
✅ **Portable** - Works anywhere with network connection
✅ **No Pi speaker needed** - Saves hardware cost
✅ **Volume control** - Use phone's volume buttons
✅ **Privacy** - Audio only plays on your phone

## Audio Settings

You can adjust audio playback in Settings:

- **Language:** French or English
- **Speech Rate:** 0.5x to 2.0x (default: 1.0x)
- **Speech Pitch:** Normal (gTTS default)
