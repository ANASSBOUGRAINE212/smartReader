# SmartReader Voice Commands Guide

## Overview

SmartReader now supports hands-free operation through voice commands! Simply tap the microphone button and speak your command.

## 🎤 How to Use Voice Commands

### 1. Activate Voice Recognition

**Tap the floating microphone button** (bottom right of screen)
- Blue microphone 🎙️ = Ready to activate
- Red microphone 🎤 = Listening
- "Listening..." badge appears when active

### 2. Speak Your Command

Speak clearly and naturally. The app will recognize your command and execute it automatically.

### 3. Deactivate

Tap the microphone button again to stop listening.

---

## 📋 Available Voice Commands

### Document Scanning

**Trigger a scan:**
- "Scan"
- "Read document"
- "Capture"
- "Take picture"

**Example:** Say "Scan" → Camera captures and processes document

---

### Playback Control

**Stop audio:**
- "Stop"
- "Pause"
- "Halt"

**Repeat last scan:**
- "Repeat"
- "Again"
- "Replay"

**Example:** Say "Stop" → Audio playback stops immediately

### Speed Adjustment

**Increase speed:**
- "Faster"

## 🔧 Setup & Permissions

### First Time Setup

When you first tap the microphone button:

1. **Permission Request** appears
2. Tap **"Allow"** to grant microphone access
3. Voice commands are now active!

### iOS Permissions

- Settings → SmartReader → Microphone → Enable
- Settings → SmartReader → Speech Recognition → Enable

### Android Permissions

- Settings → Apps → SmartReader → Permissions → Microphone → Allow

## 💡 Tips for Best Results

### Speaking Tips

1. **Speak clearly** - Enunciate your words
2. **Normal pace** - Not too fast or slow
3. **Quiet environment** - Reduce background noise
4. **Close to phone** - Within 1-2 feet
5. **Wait for beep** - Some devices beep when ready

### Command Tips

1. **Use exact phrases** - Commands listed above work best
2. **One command at a time** - Wait for execution before next command
3. **Check status** - Red microphone = listening
4. **Retry if needed** - Tap mic again if command not recognized

### Troubleshooting

**Command not recognized?**
- Speak louder or clearer
- Reduce background noise
- Try alternative phrase
- Check microphone permissions

**Microphone button not appearing?**
- Speech recognition not available on device
- Check permissions in settings
- Restart app

**Button stays red?**
- Tap again to stop listening
- Check for errors in console
- Restart app if stuck

## 🔐 Privacy & Security

### What is Recorded?

- Voice commands are processed **on-device** when possible
- Some devices may use cloud-based recognition
- Audio is **not stored** or saved
- Only command text is processed

### Permissions

- **Microphone**: Required for voice input
- **Speech Recognition**: Required for command processing

### Data Usage

- Minimal data usage for cloud recognition
- Most processing happens on-device
- No audio uploaded to SmartReader servers

## 🚀 Advanced Usage

### Continuous Listening Mode

Voice commands use **continuous listening** by default:
- Stays active until you tap button again
- Can issue multiple commands
- Automatically restarts after each command

### Command Chaining

Issue commands in sequence:
1. "Scan" → Wait for completion
2. "Faster" → Adjust speed
3. "Repeat" → Replay with new speed

### Custom Workflows

Create your own workflows:
- "Scan" + "Arabic" = Scan and read in Arabic
- "Show history" + "Search medicine" = Find medicine scans
- "Scan" + "Share" = Scan and share immediately
